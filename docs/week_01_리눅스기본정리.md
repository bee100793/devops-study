## 🔑 핵심 명령어 정리
 **위치** 
- `pwd` | print working directory | 나 지금 어디지?  
-  `ls` | list | 여기 리스트보여줘 (`-al`은 자세히) ('-R' 은 깔끔히 간략하게) 
- `cd` | change directory | 방 체인지 (`..`은 한단계 밖으로, `~`는 내방 메인으로)

**생성**
- `mkdir` | make directory | 폴더 만들기 
- `touch` | touch | 파일 만들기  

**조작**
- `cp` | copy | 원본 남기고 복사하기 (`-r`은 폴더째로) // mv [복사할것] [현재위치에서부터 옮기는곳 위치]/[사본이름]
- `mv` | move | 원본 자체를 이동하기 // mv [옮길것] [현재위치에서부터 옮기는곳 위치]/[새이름]
- `mv` |      | 이름바꾸기 / mv [현재파일명] [변경한파일명] 
- `rm` | remove | 삭제하기 (`-r`은 폴더 삭제, 복구 불가🚨)

 **권한**  
- `sudo` | superuser do | "관리자 권한으로 실행해!" (대장 모자) |
-`sudo apt-get update 
-`sudo apt-get install 

 **내용쓰기**`
 -`nano` |     | 메모장 // nano [볼 파일명] 

 **파일내용보기**
 -`cat` | concatenate | 파일 내용 다보기 
 -'less`|             | 파일 한페이지씩  보기
 -`head`|             | 앞 10줄 보기
 -`tail`|             | 밑 10줄보기 (로그 확인용)
